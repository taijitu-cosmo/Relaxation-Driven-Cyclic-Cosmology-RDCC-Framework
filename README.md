# Relaxation-Driven Cyclic Cosmology (RDCC) Framework

**A minimal CPT-symmetric one-parameter extension of ΛCDM**

RDCC is a complete, open-source theoretical and numerical framework with only one free parameter α ≈ 0.0176 ± 0.0021 (from MCMC fits). It is based on global CPT symmetry in a bipartite Hilbert space.

### Repository Structure

- `/Core/` → Central papers (Flagship, Summary, Numerical Fits, Euclid Interface) — both PDF and LaTeX source
- `/Companions/` → All technical PDF Companions (I–CI)
- `/Companions_tex/` → All LaTeX source files of the Companions
- `/Speculative/` → Conceptual and non-predictive extensions
